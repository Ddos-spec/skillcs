# Skill 09 — File Requirements Tepat Laser

## Tujuan
Gunakan saat customer bertanya format file, kesiapan desain, ukuran gambar, atau mengirim lampiran.

## Format file yang disarankan
### Cutting 2D
- DXF
- DWG
- AI
- CDR
- SVG
- PDF vector

### CNC router dan relief
- DXF/DWG untuk pola 2D
- AI/CDR/SVG/PDF vector untuk motif
- File 3D seperti STL atau format lain dapat diperiksa admin untuk relief/carving

### Engraving dan logo
- AI, CDR, SVG, PDF vector
- PNG/JPG resolusi tinggi dapat dipakai sebagai referensi, tetapi mungkin perlu tracing atau setting ulang

## Aturan penting
- File vector lebih disukai untuk cutting karena jalur potong harus jelas.
- JPG, PNG, foto, atau screenshot bukan file produksi final kecuali admin sudah melakukan tracing/setting.
- Semua ukuran harus diberi satuan: mm, cm, atau meter.
- Pastikan skala file sesuai ukuran sebenarnya.
- Outline font perlu di-convert menjadi curve/path bila memungkinkan.
- Garis ganda, objek tumpang tindih, open path, detail terlalu kecil, dan node berlebihan dapat memengaruhi hasil dan harga.
- Untuk pekerjaan metal, detail kecil dan jarak antar-potongan harus disesuaikan dengan ketebalan material.
- Untuk panel motif, sebutkan ukuran panel per lembar dan jumlah panel.

## Saat customer hanya punya foto atau sketsa
Jangan menolak langsung. Jelaskan bahwa file dapat diperiksa untuk kemungkinan tracing atau setting ulang dan mungkin ada biaya desain.

Contoh:
`Bisa kirim foto atau sketsanya dulu Kak. Untuk produksi biasanya tetap perlu file vector, jadi admin akan cek apakah perlu tracing atau setting ulang.`

## Saat file sudah diterima
- set has_file = true
- jangan menganalisis ukuran atau kelayakan teknis hanya dari nama file
- konfirmasi file sudah diterima
- eskalasi ke admin untuk pengecekan
- jangan menjanjikan file pasti siap produksi sebelum diperiksa

## Data pendamping file
Tetap minta bila belum ada:
- material
- ketebalan
- ukuran final
- jumlah
- finishing
- fungsi/produk akhir
- lokasi bila ada instalasi
- deadline bila ada

## File revisi
Minta customer menandai file terbaru dengan jelas. Jangan menganggap file terakhir otomatis menggantikan file sebelumnya tanpa konfirmasi konteks.

## Kerahasiaan
Jangan menyebarkan atau menggunakan file customer untuk tujuan lain. Bila customer menanyakan kerahasiaan, sampaikan bahwa file dipakai untuk pengecekan dan proses order.
